### Classe Autorreferencial:
Uma classe autorreferencial é uma classe que aponta para uma classe do seu mesmo tipo, uma classe dentro de uma classe. O a classe "filha" é declarada dentro da classe "mãe".
![[Pasted image 20250929154451.png]]
Cada classe aponta para uma próxima.

Existem 4 principais estruturas de dados que usam essa Autorreferencia: Lista, pilha, fila e a matriz. A pilha e a lista são bem parecidas com a lista, que é a principal, porém possuem métodos de inserção e remoção diferentes. Na pilha o primeiro a entrar é o ultimo a sair, a na fila o primeiro a entrar é o primeiro a sair.
